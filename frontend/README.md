## 💡 About

This is the Frontend part of the Quiz App . So no business logic is here.

### 🧱 Built With

1. React
2. npm
3. Tailwind CSS

### ⚙️ Local Setup

1. Setup npm in frontend root:
   ```shell
   npm install
   ```
2. Create a .env in frontend root. Follow template.env

## 👟 Usage

### 🏠 Local Usage

1.  Run client:
    ```bash
    npm start
    ```
